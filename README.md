# Sobre o projeto

O BusquePet é um aplicativo web  desenvolvido para facilitar a busca por profissionais da área veterinária. 
Usuários que possuem animais de estimação e que sentem necessidade de uma ferramenta que busque os profissionais veterinários mais bem avaliados, estabelecimentos e clínicas, encontrarão o BusquePet como a melhor alternativa. 
O aplicativo utiliza-se de ferramentas de geolocalização para busca, além de acessibilidade para pessoas que usam Libras como auxílio para comunicação.

Link do projeto: (https://busquepet.netlify.app/)
